## 1. 화면 표기 규칙

UI 요소별 표기 방식을 통일한다.

### 1-1. 대괄호 `[ ]` 로 표기하는 요소

| 요소 | 예시 |
|------|------|
| 버튼 | [선택] 버튼을 클릭 |
| 메뉴 | [사용자 필드] 메뉴를 클릭 |
| 탭 이름 | [보안] 탭을 클릭 |
| 입력 필드/레이블 | [이메일] 입력란에 값을 입력 |
| 단축키 | [Ctrl] + [C] |
| 셀 | [C2] 셀, [A5] 셀 |
| 열 | [A:B] 열 |
| 범위 | [C2:C75] 범위 |
| 키 | [Enter] 키 |
| 대화상자 | [Preference] 대화상자에서~ |
| 그룹·패널·레이어 이름 | [Edit] 그룹 |
| 아이콘·버튼(그림 전용) | ㅇㅇ(이미지) 아이콘, [ㅇㅇ] 버튼 |

> 단축키는 `[Ctrl] + [C]`처럼 `+` 기호로 연결한다.

### 1-2. 작은따옴표 `' '` 로 표기하는 요소

| 요소 | 예시 |
|------|------|
| 클릭 가능한 텍스트(링크/클릭 이벤트) | '텍스트'를 클릭 |
| 옵션값 | [Transform] 속성을 '5'로 수정합니다. |
| 체크박스 | '자동 저장' 옵션을 선택 |
| 라디오 버튼 | '남성'을 선택 |
| 토글 | '다크 모드' 토글을 활성화 |
| 기본값·변수값 강조 | 기본값은 '8'입니다. length 변수를 '10'으로 수정합니다. |
| 강조할 부분 또는 오류 메시지 | '계산할 수 없음' 오류 메시지가 나타납니다. |

### 1-3. 큰따옴표 `" "` 로 표기하는 요소

| 요소 | 예시 |
|------|------|
| 화면/창 이름 | "What best describes you?" 화면에서 |

| 요소 | 표기 방법 | 예시 |

## 2. 띄어쓰기

### 2-1. 의존 명사

**원칙:** 앞말이 관형사형 활용형이면(수식을 받으면) → 띄어 씀 / 조사·접사이면(수식 안 받으면) → 붙여 씀

| 단어 | 조사(붙여 씀) | 의존 명사(띄어 씀) |
|------|--------------|------------------|
| 뿐 | 남자뿐, 교수뿐 아니라 | 웃을 뿐, 교수일 뿐만 |
| 대로 | 법대로, 약속대로 | 아는 대로, 본 대로 |
| 만큼 | 전봇대만큼, 별색만큼 | 볼 만큼, 노력한 만큼 |
| 만 | 하나만 알고 | 사흘 만에 |
| 치고 | 공대생치고 | 그런 셈 치고 |
| 들 | 학생들(접미사) | 쌀, 보리, 기장 들을(의존 명사) |
| 차 | 확인차 들르다(접미사) | 고향에 갔던 차에(의존 명사) |
| 지 | 큰지 작은지(어미), 얼마나 비싼지(어미) | 만난 지, 종사한 지(시간 경과) |
| 데 | 헤어질 결심하라는데(어미) | 헤어질 결심하는 데 준비가 필요해, 먹는 데 방해(곳·경우 의미) |
| 듯 | 말했듯이(어미) | 뛸 듯이 기쁘다, 취소된 듯하다(의존 명사) |
| 판 | 노름판, 씨름판, 웃음판(합성어) | 바둑 한 판 두자(의존 명사) |
| 것, 바, 따위 | — | 의존 명사이므로 모두 띄어 씀 |
| 대신 | — | 오류 대신, 노는 대신(의존 명사) |
| 간 | 이틀간, 수년간, 3개월간(접미사 → 붙여 씀) | 부모와 자식 간, 사제 간(의존 명사 → 띄어 씀) |

> **참고:** 부부간·부자간·부녀간처럼 한 단어로 굳어진 단어는 붙여 씀. 교정 시 사전 검색 필요.

### 2-2. 보조 용언

⚠️ **충돌 항목 — 1팀 규칙 우선 적용**

| 규칙 | 1팀 교정 규칙 | 통일안 | 다산북스 |
|------|-------------|--------|---------|
| 보조 용언 기본 원칙 | 본용언과 보조 용언은 **띄어 씀** | 보조 용언은 **붙여 씀** | 띄어 씀 원칙, 일부 붙여 씀 허용 |

> **적용 기준:** 1팀·다산북스 규칙에 따라 본용언과 보조 용언은 원칙적으로 **띄어 씀**.  
> 단, 아래 허용 조건에 해당하거나 사전에 합성어로 등재된 단어는 붙여 씀.

**다산북스 붙여 씀 허용 조건 (2가지):**

**① '본용언 + -아/-어 + 보조용언' 구성** → 붙여 쓰는 것을 허용

| 원칙(띄어 씀) | 허용(붙여 씀) |
|-------------|-------------|
| 먹어 보았다 | 먹어보았다 |
| 꺼져 간다 | 꺼져간다 |
| 막아 낸다 | 막아낸다 |
| 도와 드린다 | 도와드린다 |
| 깨뜨려 버렸다 | 깨뜨려버렸다 |

**② '형용사 + -아/-어하다' 구성** → 보조용언 '하다' 붙여 씀

| 올바른 표기 | 틀린 표기 |
|-----------|---------|
| 예뻐한다(○) | 예뻐 한다(×) |
| 행복해하다(○) | 행복해 하다(×) |

**③ '동사 + -아/-어 하다' 구성** → 반드시 띄어 씀

| 올바른 표기 | 틀린 표기 |
|-----------|---------|
| 못 잊어 한다(○) | 못 잊어한다(×) |

**띄어 쓰는 경우 (원칙):**

| 예시 | 비고 |
|------|------|
| 비어 있다 | 있다 앞은 무조건 띄어 씀 |
| 서 있다, 들어 있다 | 있다 앞은 무조건 띄어 씀 |
| 아는 척하다 | 척하다 앞 띄어 씀 |
| 막아 내다 | |
| 사라져 버리다 | |
| 생각해 보다 | |
| 매달아 놓다 | |
| 알려 주다 | |
| 옮겨 주다 | |
| 가르쳐 주다 | |
| 그린 듯한, 쓸 만하다, 있을 법한 | 듯하다, 만하다, 법하다 앞 띄어 씀 |
| 책을 읽다 보면 | -하다 보면 띄어 씀 |
| 사랑했었나 봐 | -나, -까 뒤 보조 용언 띄어 씀 |
| 쌓아 올리다, 따라 하다, 붙여 넣다 | 본동사+본동사 형태 띄어 씀 |
| 도와 드리다 | |
| 기어코 성공하고야 말겠다 | 고야 말다 |

**붙여 쓰는 경우 (합성어로 사전 등재):**

살펴보다, 알아보다, 알아두다, 가져오다, 도와주다, 알은체하다, 다시없다, 마지않다, 못쓰다, 해보다, 궁금해하다, 내려받다, 해놓다

#### 다산 1장. 보조용언의 띄어쓰기

⚠️ **1팀 교정 규칙 우선 적용** — 세부 패턴은 다산북스 기준 참고

| 구성 | 규칙 | 예시 |
|------|------|------|
| 본용언 + -아/-어 + 보조용언 | 띄어 씀 원칙, 붙여 씀 허용 | 먹어 보았다 → 먹어보았다 / 도와 드린다 → 도와드린다 |
| 형용사 + -어하다 | 붙여 씀 | 예뻐한다(○) / 예뻐 한다(×) / 행복해하다(○) |
| 동사 + -아/-어 하다 | 띄어 씀 | 못 잊어 한다(○) / 못 잊어한다(×) / 못 견뎌 했다(○) |
| 구(句) + -아/-어 하다 | 띄어 씀 | 먹고 싶어 하다(○) / 마음에 들어 하다(○) / 내키지 않아 하다(○) |
| -어 있다 | 띄어 씀 | 깨어 있다 / 앉아 있다 / 꽃이 피어 있다 |
| 조사가 붙는 경우 | 띄어 씀 | 잘도 놀아만 나는구나 / 책을 읽어도 보고 / 잘난 체를 한다 |

**본용언이 복합어인 경우 — 활용형 음절 수 기준:**

| 활용형 음절 | 규칙 | 예시 |
|-----------|------|------|
| 2음절 이하 | 붙여 씀 허용 | 해주다 / 웃겨보아라 / 나가버렸다 / 말해달라고 |
| 3음절 이상 | 반드시 띄어 씀 | 쫓아내 버렸다(○) / 매달아 놓는다(○) / 집어넣어 둔다(○) |

> **예외:** '만들어 보다', '가르쳐 주다'는 활용형이 3음절이나 '만들다', '가르치다'가 복합어가 아니므로 붙여 씀 허용 → 만들어보다, 가르쳐주다

**관형사형 + 보조용언(의존명사 + -하다/-싶다) 구성 — 다산북스는 반드시 띄어 씀:**

| 올바른 표기(다산) | 허용 표기(한글 맞춤법) |
|----------------|------------------|
| 아는 체하다(○) | 아는체하다 허용 |
| 비가 올 듯하다(○) | 비가 올듯하다 허용 |
| 그 일은 할 만하다(○) | 할만하다 허용 |
| 일이 될 법하다(○) | 될법하다 허용 |
| 비가 올 성싶다(○) | 올성싶다 허용 |

**보조용언이 거듭 나타나는 경우 — 앞의 보조용언만 붙여 씀 허용:**

적어 둘 만하다 → 적어둘 만하다 / 읽어 볼 만하다 → 읽어볼 만하다
